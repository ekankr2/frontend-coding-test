# var, let, const 차이점

### var

- 중복 선언 가능
- 재선언, 재할당 가능으로 인해 문제 변수 위치 파악이 힘들다는 단점이 있음.
- 해결을 위해 ES6부터 let, const 도입

### let

- 중복 선언 불가, 재할당 가능

### const

- 불변(immutable)
- 재선언, 재할당 불가

[changing Const Object](Javascript/letConst.js)
